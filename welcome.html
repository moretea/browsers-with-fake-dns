Validating DNS settings

To manually inspect how the DNS in this container is configured, open a terminal (right click on desktop -> terminal), and run a command like dig $TLD ANY.

Updating DNS records

Update the zone.yaml file outside of this container. The DNS settings will be automatically rebuild.
